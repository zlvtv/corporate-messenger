# TeamBridge

Платформа для командной работы, объединяющая чат и управление задачами. Позволяет обсуждать идеи в реальном времени и мгновенно превращать сообщения в задачи с контролем исполнения.

## Возможности

- **Аутентификация с подтверждением по email**
- **Восстановление пароля через email**
- **Организации и проекты с иерархией доступа**
- **Чат в реальном времени** (Firebase Realtime)
- **Создание задач из сообщений** (правый клик → "Сделать задачей")
- **Доска задач** с фильтрацией, статусами и сроками
- **Календарный вид задач**
- **Реальное время** (Firebase Realtime + onSnapshot)
- **Адаптивный интерфейс** 

## Технологии

- **Frontend**: React, TypeScript, Vite
- **Состояние**: Context API
- **Бэкенд**: Firebase (Authentication, Firestore, Realtime)
- **Деплой**: Render
- **Пакетный менеджер**: pnpm

## Архитектура

Фронтенд построен на модульной системе с разделением по контекстам:  
`AuthContext`, `OrganizationContext`, `ProjectContext` — обеспечивают прозрачный доступ к данным на всех уровнях приложения.

## Безопасность
- Пароли хранятся в хэшированном виде
- Аутентификация через JWT
- Ограничение попыток входа
- XSS-фильтрация контента
- RLS в PostgreSQL
- Пароли хранятся в Firebase (хэшированы)
- Аутентификация через Firebase Auth (JWT)
- Ограничение попыток входа (встроено в Firebase)
- XSS-фильтрация контента (через DOMPurify)
- Подтверждение email перед входом в систему

## Лицензия
Исходный код доступен для использования в образовательных целях с указанием авторства.
