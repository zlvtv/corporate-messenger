## Важные замечания
1. После переименования необходимо обновить все импорты в проекте.
2. Рекомендуется выполнять переименование по одному компоненту с последующим тестированием.
3. Используйте систему контроля версий (git) для фиксации изменений после каждого успешного шага.
4. Некоторые компоненты уже соответствуют БЭМ-названиям и не требуют изменений.
5. Для сложных компонентов рекомендуется разбить CSS на элементы (__element) и модификаторы (_modifier).
